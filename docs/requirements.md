# 機能要件

## 1. 概要
本システムは、HTTP リクエストで呼び出せる 2 つの API（掛け算・割り算）を提供する。
ブラウザで URL を入力してアクセスした際に、計算結果をレスポンスとして画面に表示する。

## 2. 提供機能
- 掛け算 API を提供する。
- 割り算 API を提供する。

## 3. API 仕様
### 3.1 共通仕様
- HTTP メソッドは GET とする。
- 入力値はクエリパラメータ A, B とする。
- A, B は数値として解釈可能であること。
- API 呼び出し結果は HTTP レスポンスとして返却する。

### 3.2 掛け算 API
- 処理内容: A × B を計算する。
- 正常時: 計算結果を返す。

### 3.3 割り算 API
- 処理内容: A ÷ B を計算する。
- 正常時: 計算結果を返す。
- 異常時: B = 0 の場合、400 Bad Request を返す。

## 4. 入出力要件
- 入力は URL クエリ文字列で受け付ける。
- レスポンスはブラウザで確認可能な形式で返す。

## 5. エラーハンドリング
- 必須クエリパラメータ A, B が不足している場合は 400 Bad Request を返す。
- A, B が数値として不正な場合は 400 Bad Request を返す。
- 予期しない内部エラーは 500 Internal Server Error を返す。
